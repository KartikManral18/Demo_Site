Frontend Development

Front-end Development is the development or creation of a user interface using some markup languages and other tools. It is the development of the user side where only user interaction will be counted. It consists of the interface where buttons, texts, alignments, etc are involved and used by the user,through the use of HTML, CSS and Javascript

The Three Bacis Languages Used in Frontend Development :-

  • HTML
  • CSS
  • Javascript

  • HTMLis a standard markup language used to create web pages. It is the backbone of a websiteâ€™s structure and content, allowing developers to organize and format text, images, and other multimedia elements.

  • CSS :-
    1. CSS stands for Cascading Style
    2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media
    3. CSS saves a lot of work. It can control the layout of multiple web pages all at once
    4. External stylesheets are stored in CSS files.

  • Javascript:- JavaScript is used to add dynamic interactions to web pages, such as animated graphics, interactive maps, and timely content updates. It works with HTML and CSS to complement CSS's formatting capabilities with user interaction
